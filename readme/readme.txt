------------------------------------------------------
Kangle虚拟主机管理系统安装教程(以下简称：kangle vhms)
------------------------------------------------------
--------------------------
一．kangle vhms 系统简介
--------------------------
kangle vhms是一款专为虚拟主机商开发的虚拟主机管理系统;
kangle vhms全自动化开设虚拟主机！支持多种动态语言，系统采用php+mysql开发;
节点使用c++开发，安装成kangle的一个模块，通过whm接口和管理系统通信;
支持磁盘配额！mysql数据库自动开通管理，ftp和在线文件管理。
kangle vhms支持在线一键安装程序。


-----------------------------
一．kangle vhms 安装环境需求
-----------------------------
1．需安装pdo组件和php(5.2以上版本) + mysql(5.0以上版本)
2．php需安装pdo和pdo_mysql这两个组件
3．mysql需安装innodb数据库存引擎

-----------------------------
二．kangle vhms 开始安装
-----------------------------
1. 上传 upload 目录中的文件到服务器
2. 设置文件权限, 目录需要写入的权限
3. 执行安装脚本, 请在浏览器中运行 install 程序，即访问 http://您的域名//install.php
4. 参照页面提示，进行安装，直至安装完毕

-----------------------------
三. 登陆kangle vhms后台管理
-----------------------------
进入kangle vhms后台管理，在IE地址栏输入 http://您的域名/admin/  登陆后就可以开始管理您的帐户了

-----------------------------
四. 增加节点服务器
-----------------------------
请参考有关kanlge虚拟主机管理系统节点的安装文档

--------------------------------
五. kangle vhms 增加虚拟主机产品
--------------------------------
登陆kangle vhms管理后台，在页面左下角点 虚拟主机产品-->增加产品, 然后就可以按您的需求增加您的产品信息了

-----------------------------
六. kangle vhms 用户注册
-----------------------------
用户注册地址：http://您的域名/user/frame.php?c=public&a=registerForm
您可以在您的网站上嵌入该网址供用户注册
