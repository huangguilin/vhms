一.新手指南

  • 如何注册使用空间?

    • 答：注册会员帐号后登陆会员中心点击购买产品列表，或者在顶部右边的产品快速通道中选择产品注册购买;

  • 请问我如何付款

    • 目前我们为您提供的支付方式有：网上付款、银行电汇、到我公司付款,支票付费等付款方式,您可以择其一种方便您付款的方式进行支付 ，支付后可登陆会员中心查看你的余额，如余额未到账，请查看您的充值记录并及时联系我们。

      二.虚拟主机相关问题
  • 空间如何上传文件?

    • 答：使用FTP传输工具，或者在管理面板中使用在线文件管理功能上传,如果文件太多，可压缩后上传，我们支持在线解压缩


  • 请问你们的在线安装程序，可以放上我常用的程序吗

    • 可以的，只要你的程序是安全的，安装程序可以定制


  • 请问我安装了在线安装程序里的discuz程序，我要换别的怎么办

    • 您好，通过在线安装程序里安装的程序，如果您需要更换，可以在安装程序里删除再安装其它程序即可。删除之前请备份数据库

  • 请问 你们的产品可以定制吗
    • 您好，我们的产品大小如果您不满意，可以联系我们客服，定制您需要的产品大小。
  • 为什么我上传的rar文件无法解压

    • 答：你好，我们的管理系统支持7z和zip格式的文件解压，暂时不支持rar文件解压


  • 请问我的网站如何备案

    • 你好，请登陆http://www.miitbeian.gov.cn备案网站，注册后填写网站资料，等待信息产业部审核即可。


  • 我遇到问题如何寻求帮助？

    • 当您在使用我们产品时遇到问题，您可以通过以下方式获得帮助

    1.首先访问我们首页上方帮助中心，查找相应解决方法，强烈建议您使用“输入问题找出答案”功能，即可搜索到所有关于的说明 ；


    2.如果您无法在客服中心频道中寻找到相应解决方法时，可以登陆我们会员区，在线提交问题，我们工作人员将在第一时间为您解答；

    3.您也可以随时拨打24小时客户服务热线，我们客服将竭诚为您服务
